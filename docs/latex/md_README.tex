This repo contains a C++ implementation of a templated, double-\/linked, list using smart pointers.

\doxysection*{Documentation}

You can find a Doxygen generated documentation \href{https://fdrmrc.github.io/TemplatedDoubleLinkedList/}{\texttt{ here}}.

\doxysection*{How to compile and run}

First, clone the repo in your local machine with


\begin{DoxyCode}{0}
\DoxyCodeLine{git clone https://github.com/fdrmrc/TemplatedDoubleLinkedList.git}
\end{DoxyCode}


then move in the repo folder and type


\begin{DoxyCode}{0}
\DoxyCodeLine{make}
\end{DoxyCode}


and


\begin{DoxyCode}{0}
\DoxyCodeLine{./exe.x}
\end{DoxyCode}


to execute.

\doxysection*{Check memory leaks with {\bfseries{valgrind}}}

To check if the code leaks memory\+:


\begin{DoxyItemize}
\item Use Makefile\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{make}
\end{DoxyCode}

\item Execute using {\itshape valgrind} (requires \href{https://valgrind.org}{\texttt{ valgrind}}) 
\begin{DoxyCode}{0}
\DoxyCodeLine{valgrind ./exe.x}
\end{DoxyCode}
 
\end{DoxyItemize}
